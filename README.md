"# Fullstack-Laravel-and-vue-js"

## backend

```
cd users-departments_backend

composer install

cp .env.example .env

```

### config database

```
php artisan migrate
php artisan generate:key
php artisan serve
```

## unit test

```
php artisan test
```

## frontend

```
cd users-departments_frontend
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
